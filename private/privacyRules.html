Politique de confidentialité de CyberProtect

CyberProtect est une extension Chrome conçue pour améliorer le bien-être numérique en modérant les contenus toxiques sur les plateformes sociales.

Nous ne collectons, stockons ni transmettons aucune donnée personnelle ou information utilisateur.

Toutes les analyses de contenu se font localement ou via des API tierces (comme CommentAnalyzer) sans identification de l'utilisateur.

Aucune donnée personnelle n’est enregistrée, transmise, ni partagée avec des tiers.

Pour toute question, vous pouvez nous contacter à : yourcyberprotect@gmail.com
